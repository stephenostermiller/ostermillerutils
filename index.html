Installation

The utilities are pre-compiled and ready to use. Simply use one of the following to add utils.jar to your classpath:

  • Put utils.jar in the Java extensions directory.
  • Set your CLASSPATH environment variable to include utils.jar.
  • Specify utils.jar in the command line classpath when you compile and run your programs.
  • Extract utils.jar to a folder already in your classpath.
  • Include the contents of utils.jar in your program's jar file.
  • Reference utils.jar from the manifest file in your program's jar file.
You can extract the contents of utils.jar either with the jar tool that comes with java or by renaming utils.jar to utils.zip and using any zip file extractor.

If you are running unix and you want to have easy access to the LinEnds, RandPass, and MD5 command line programs that come in these utilities, run this install shell script.

Java Utilities

Download the latest version of these utilities including source code. (Jar - 322k)

[Browse the source code. | View the documentation.]

  • Base64 - Encode and decode base 64.
  • Browser - Open a browser from an application on Windows, Unix, or Macintosh.
  • CGI Query String Parser - Libraries to to parse the query data supplied by HTTP GET or POST requests.
  • Circular Byte Buffer - Implements the circular buffer producer/consumer model for bytes.
  • Circular Character Buffer - Implements the circular buffer producer/consumer model for characters.
  • Circular Object Buffer - Implements the circular buffer producer/consumer model for Objects.
  • Comma Separated Values (CSV) - Libraries to read and write files in Comma Separated Value (CSV) format.
  • Comma Separated Values (CSV) Excel Format - Libraries to read files in Comma Separated Value (CSV) format exported from Excel spreadsheet.
  • Line Ends - Converts text files between various system formats.
  • MD5 Sums - Classes for generating MD5 checksums on String, Byte arrays, Streams, or Files.
  • Password Dialog - A password dialog box written for Swing.
  • Random Password Generator - Generate a random String using a cryptographically secure random number generator.
  • Significant Figures - Handles parsing, rounding, and displaying numbers for scientific applications.
  • String Helper - Methods for formatting, manipulating, and querying Strings.
  • String Tokenizer - A StringTokenizer replacement with enhancements.
  • Straight Stream Reader - An input stream reader that does no character class translation.

Modifications

Several tools are useful if you want to modify and recompile these utilities:
  • Java SDK - Includes the Java compiler.
  • JFlex - to generate lexers used by several utilities.
  • Make - to automate the build. Consider "make", "make clean", "make build", and "make docs".
  • Cygwin - to get unix utilities (including make) on Windows.
  • CVS - to check out the latest version of these utilities from the Giant Java Tree.
